Hướng dẫn cài đặt MikroTik

Dựa trên file config.json, hệ thống đã tạo sẵn script riêng cho router của bạn.

1. Script thu thập dữ liệu (Chạy mỗi 5 phút)

Tạo một Script mới trên MikroTik tên là traffic-collector và dán đoạn code này vào:

Vào System -> Scheduler, tạo 1 scheduler chạy mỗi 5 phút: /system script run traffic-collector

2. Script chốt sổ ngày (Chạy lúc 00:01)

Tạo Script mới tên là daily-traffic-report và dán đoạn code này vào:

Vào System -> Scheduler, tạo 1 scheduler chạy vào lúc 00:01:00 mỗi ngày: /system script run daily-traffic-report
